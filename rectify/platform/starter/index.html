Questions

Do visitors see the editor?

No. The overlay is injected only for the logged-in owner — everyone else gets clean static markup.

Does it edit the live DOM or my files?

Your actual source files under your site directory. The changes are real, on disk, and reversible.

Which frameworks work?

Plain HTML, React, Vue, and Svelte — it locates source by reading your files, not by relying on a build plugin.

Which AI model does it use?

Any LiteLLM-supported model: Anthropic Claude, OpenAI GPT, or a local Ollama model, chosen with an env var.

How do I deploy it?

One-click on Railway, any Docker host, or python -m rectify.platform locally.

Is my data sent anywhere?

Only your edit prompts go to the model you choose. The site lives on your server — there's no SaaS backend.
